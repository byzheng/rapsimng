# Set the log level of apsimx file

Set the log level of apsimx file

## Usage

``` r
log_level(l, level = c("Error", "Warning", "Information", "Diagnostic", "All"))
```

## Arguments

- l:

  the list of apsimx file

- level:

  log level with option Error, Warning, Information, Diagnostic, All

## Value

a new apsimx file
