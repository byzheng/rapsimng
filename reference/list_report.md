# List all reports in the database

List all reports in the database

## Usage

``` r
list_report(file)
```

## Arguments

- file:

  the file path to apsimx or db file

## Value

a vector of all reports

## Examples

``` r
if (FALSE) { # \dontrun{
file <- system.file("extdata/wheat.apsimx", package = "rapsimng")
list_report(file)
} # }
```
