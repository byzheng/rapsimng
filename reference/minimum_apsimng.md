# Create the minimum requirements to run an APSIM Next Generation

Create the minimum requirements to run an APSIM Next Generation

## Usage

``` r
minimum_apsimng(install_path, output)
```

## Arguments

- install_path:

  The installed path of APSIM Next Generation

- output:

  The output folder

## Examples

``` r
if (FALSE) { # \dontrun{
 minimum_apsimng("C:/ProgramFiles/APSIMNG", "minimum_apsimng")
} # }
```
